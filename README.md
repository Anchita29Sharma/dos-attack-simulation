# DoS Attack Simulation and Analysis

This project demonstrates the simulation and analysis of two Denial of Service (DoS) attacks using Python and Wireshark.

## Attacks Implemented
1. TCP SYN Flood Attack
2. ICMP Flood Attack

## Tools Used
- Python
- Scapy
- Wireshark

## Description
The attacks were simulated using Python scripts that generate network packets using the Scapy library. Wireshark was used to capture and analyze the generated traffic.

## Files Included
- syn_flood.py – Python script for SYN Flood attack
- icmp_flood.py – Python script for ICMP Flood attack
- network_assignment.pdf – Detailed report of the experiment
- Screenshots of packet captures and analysis
